Compute the Vista+ ESE XOR page checksum.

Seeds with the logical page number (physical_page_number - 1), then XORs every 4-byte word across the entire page (including bytes[0..8]). For a correctly checksummed page the result equals the stored XOR value at bytes[4..8]. Data beyond the last full 4-byte word is ignored.
