pub fn detect_timestamp_skew(
    header: &EseHeader,
    db: &EseDatabase,
) -> Vec<EseStructuralAnomaly>
Expand description

Compare each page’s db_time field (page offset 0x08, 4 bytes) against the file header’s db_time low 32 bits.

A page whose db_time is greater than the header’s indicates the page was written after the header was last updated — a manipulation indicator.
