Expand description

A raw structural anomaly detected in an ESE database binary.

These are parsing-level observations, not forensic conclusions.

Variants§

§

DirtyDatabase

Database was not cleanly shut down (state = 2).

Fields

§db_state: u32
§

TimestampSkew

A page’s db_time field is strictly greater than the file header’s db_time low 32 bits — the page appears newer than the header.

Fields

§page_number: u32
§header_db_time_low: u32
§page_db_time: u32
§

SlackRegionData

Non-zero bytes exist in the page slack region (between record data end and tag array start) — possible residual record fragments.

Fields

§page_number: u32
§offset_in_page: u16

Byte offset within the page where slack begins.

§length: u16

Number of non-zero slack bytes.

§

PageChecksumMismatch

The XOR or ECC checksum stored in the page header does not match the recomputed checksum over page bytes. Bytes changed after write.

Fields

§page_number: u32
§expected: u32
§actual: u32
§

BTreeLinkBroken

A B-tree node’s sibling-page pointer chain is broken: the declared next/previous page does not reciprocate the link.

Fields

§page_number: u32
§broken_sibling: u32

Page number that was supposed to link back.

§

PageFlagInconsistency

A page is reachable via the B-tree but its page_flags are inconsistent with its position (e.g. leaf flag set on an internal node).

Fields

§page_number: u32
§flags: u16
§context: &'static str
§

OrphanedSrumTable

A SRUM table identified by GUID in the catalog is referenced by a record but no corresponding B-tree root page can be found.

Fields

§table_guid: String
§

MissingSrumTable

A required SRUM table (known GUID from forensicnomicon) is absent from the catalog — the table was deleted or never populated.

Fields

§table_guid: &'static str
§table_name: &'static str
§

TruncatedDatabase

The file ends before the declared page count implies. The database was truncated — either during acquisition or deliberately.

Fields

§declared_pages: u32
§actual_pages: u32
§

OrphanedCatalogEntry

A catalog entry references a B-tree root page that falls beyond the declared page count (i.e. the page does not exist in the file).

This can indicate a partially written or deliberately truncated database where the catalog was updated but the corresponding data pages were not.

Fields

§object_name: String

Name of the catalog entry (table or index name).

§declared_page: u32

The root page number that the catalog declares.

§last_valid_page: u32

The highest valid page index in the file.

§

AutoIncIdGap

One or more AutoIncId values are missing from a contiguous sequence.

A gap between prev and next (non-adjacent integers) indicates that records were deleted without leaving a deleted-tag marker — either via bulk deletion or external manipulation of the ESE store.

Fields

§prev: i32

The AutoIncId immediately before the gap.

§next: i32

The AutoIncId immediately after the gap.

§

DeletedRecordPresent

A record tag has the deleted-record flag set (bit 29 of the tag word).

Deleted records are not immediately zeroed; the raw bytes often persist and can contain recoverable data or evidence of prior state.

Fields

§page_number: u32
§tag_index: usize

0-based tag index within the page’s tag array.
