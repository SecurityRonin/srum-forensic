PushNotificationRecord binary decoder.

Decodes the synthetic 24-byte wire format used in test fixtures.

Binary layout (all little-endian):

  • [0..8]: filetime (u64)
  • [8..12]: app_id (i32)
  • [12..16]: user_id (i32)
  • [16..20]: notification_type (u32)
  • [20..24]: count (u32)
