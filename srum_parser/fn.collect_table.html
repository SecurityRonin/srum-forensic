) -> Result<Vec<T>>
Expand description

Iterate a named ESE table and decode each record, silently skipping records that fail either the ESE read or the domain decode step.

Returns Ok(vec![]) when the table is absent — correct behaviour for Windows Server 2022 which omits several SRUM extensions entirely.
